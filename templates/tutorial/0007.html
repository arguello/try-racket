Now that we've defined our (guess) function; we need to write the functions for smaller and bigger.

Like guess, these functions are defined with the define form. Let's start with smaller:

(define (smaller)
  (set! upper (max lower (sub1 (guess))))
  (guess))

Like (guess), we wrap the function name smaller in parens because it takes no parameters.

Second, the function body consists of two expressions, one per line.

Third, the function uses a set! expression to change the value of a variable.

The purpose of set!, pronounced set bang, is to evaluate the expression and set the variable to the resulting value.

Go ahead and type: bang bang

the bang in set bang refers to the exclamation mark: ! The term has it's roots in comics, where Bang was usually emphasized by an exclamation mark. In computing, people prefer to say the short, cool sounding word bang! over the five-syllable phrase exclamation mark.
