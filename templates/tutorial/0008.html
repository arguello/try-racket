We're going to "kill" our current upper limit. We need a newer, fresher upper limit.

Go ahead and type upper in the interaction panel.

It should return the upper limit that we set: 100

Now type: (smaller), and again type: upper

So, our weapon of choice, the set! expression in the smaller function:

(define (smaller)
  (set! upper (max lower (sub1 (guess))))
  (guess))
  • decrements the value returned by guess: (sub1 (guess))
  • uses the max function to return the largest of lower and (sub1 (guess))
  • assigns the result of max to upper, setting a new upper limit

The smaller function hammers the final nail into the coffin by placing a call to (guess).

Take some time to mourn, then type: brutal
