The sqrt function requires a parameter:

(sqrt parameter)

The term parameter refers to the variable as found in the function definition.

When we pass an actual value or expression, we call it an argument. So, in the function (sqrt 144):

  • the function name is: sqrt
  • the argument is: 144

When you pass the argument 144 to the sqrt function, the function "knows" to calculate the square root of 144.

You can also pass a function as an argument:

(sqrt (expt 12 2))

The expt function "knows" to raise it's first argument 12 to the power of it's 2nd argument: 2

The sqrt function then calculates the square root of the resulting value. Try: (expt (sqrt 144) (/ 4 2))

parameter and argument
are terms sometimes used interchangeably in programming. The term parameter is often used to refer to the variable as found in the function definition, while argument refers to the actual value passed to the function.
