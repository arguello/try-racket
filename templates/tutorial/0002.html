Funky Functions

(λ)

A function is a named program fragment that "knows" how to perform a defined task.

In (circle 10), the function name is circle, and the argument is 10. When you pass the argument to the circle function, the function "knows" to draw a 10 pixel circle.

There's also a rectangle function that takes two arguments instead of one:

(rectangle 10 20)

Try giving circle the wrong number of arguments, just to see what happens:

(circle 10 20)

In addition to basic picture constructors like circle and rectangle, there’s a function that combines pictures:

(hc-append (circle 10) (rectangle 10 20))

Racket is a full-spectrum programming language. It is a descendent of a language called Scheme, which in turn is a descendent of Lisp.
