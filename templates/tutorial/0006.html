As mentioned, our Guess My Number game will use a binary search algorithm to "guess" the number.

All that means is that the computer will "guess" the player's number by computing the average of the lower and upper limits.

To do that, we need to define a function for guessing; type this in the interactions panel:

(define (guess)
   (quotient (+ lower upper) 2))

Look complicated? Let's break it down:

  • the name of the function is: guess
    • the function won't take arguments, so we wrap the name in parentheses: (guess)
  • we add the lower and upper limits
    • we use variables instead of the values 1 and 100, so that we can change these values
  • we then divide the resulting value by the number of arguments passed to the addition function: 2

Now test your function by typing: (guess)
