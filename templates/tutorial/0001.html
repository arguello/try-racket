The panel to the right is the interactions panel. It's an example of a read-eval-print-loop, or REPL.

The interactions panel is where you'll be able to type an expression, hit Enter, and watch it run!

The first expression you typed was (+ 1 2). An expression can also be just a value, such as the number 5, or the string "art gallery":

Try typing 5 or "art gallery" in the interactions panel.

An expression can also be a function call.

To call a function, put an open parenthesis before the function name, then expressions for the function arguments, and then a close parenthesis, like this:

(circle 10)

Go ahead and type that function into the REPL.

The REPL originated in Lisp, a powerful programming language invented by Artificial Intelligence (AI) pioneer John McCarthy. This tutorial uses a descendent of Lisp called Racket.
