The panel to the right is the interactions panel. It's an example of a read-eval-print-loop, or REPL.

The interactions panel is where you'll be able to type an expression, hit Enter, and watch it run!

The first expression you typed was a value. A value can also be something as simple as the number 42.

Try typing the following in the interactions panel:

"coding is easy"

144

An expression can also be a function:

(* 2 3 4 6)

(* 2 (+ 2 4) (/ 48 2))

A function is a named program fragment that "knows" how to perform a defined task.

Type this function into the interactions panel:

(sqrt 144)

the REPL originated in Lisp
a powerful programming language invented by Artificial Intelligence (AI) pioneer John McCarthy. This tutorial uses a descendent of Lisp called Racket.
