The Main Function

It is practical to have one main function that starts - or restarts - the whole application.

This is a simple feat for Guess My Number:

(define (start n m)
  (set! lower (min n m))
  (set! upper (max n m))
  (guess))

As you now know, start takes two arguments, which are the numbers we want to set as the lower and upper limits.

By using the max and min functions, we cut down on the instructions that we need to give any player.

A player simply needs to put in any two numbers in any order, and the function can determine the lower and upper limits. Try it by typing: (start 30 1)
