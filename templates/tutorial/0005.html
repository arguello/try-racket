The define expression just set the lower limit of the player's number by defining the variable lower.

We set variables by using the define keyword.

A keyword is a word reserved by a programming language because it has a special meaning.

We follow the define keyword with the name we wish to assign the variable, and we follow the name with an expression that produces the value we want the variable to have.

In (define lower 1):

  • the name of the variable is: lower
  • the expression assigned to lower is: 1

Go ahead and type lower into the interactions panel, and it will return it's assigned value: 1

Let's set our upper limit. In the interactions panel, define a variable named upper with the value 100:

(define upper 100)
